# 深海采矿环境教育模拟 (Deep Sea Mining Environmental Simulation)

## 项目简介

这是一个基于Unity开发的深海采矿环境教育模拟项目。通过交互式模拟，展示集矿机收集多金属结核的过程，演示沉积物羽流扩散对周围生态系统的影响。玩家可以调整采集速率，实时观察各项环境指标的变化。

## 功能模块

### 1. 海底地形生成 (Seabed Terrain Generation)
- 使用Perlin噪声算法生成真实感海底地形
- 支持多八度噪声叠加，创造丰富的地形细节
- 自动生成多金属结核矿点
- 可配置的地形大小、高度、噪声参数

### 2. 集矿机控制 (Mining Machine Control)
- WASD键控制集矿机移动
- 空格键启动采矿功能
- 集矿机自动跟随地形起伏
- 可调节采矿速率(0-100%)
- 采矿范围随速率动态调整

### 3. 沉积物羽流模拟 (Sediment Plume Simulation)
- 实时粒子系统模拟沉积物扩散
- 羽流强度与采矿强度联动
- 粒子随时间自然消散
- 可配置扩散方向、速度和生命周期

### 4. 生态指标系统 (Ecology Indicator System)
- **总体环境健康度**: 综合评分(优秀/良好/一般/较差/危急)
- **水质**: 沉积物对水体的影响
- **生物多样性**: 海洋生物受影响程度
- **栖息地完整性**: 地形破坏程度
- **碳封存能力**: 生态系统碳汇功能

### 5. 海洋生物模拟 (Marine Life Simulation)
- 5种海洋生物: 鱼类、珊瑚、虾、水母、海绵
- 生物对沉积物有压力反应
- 高压力下生物会逃离或变色
- 压力降低时逐渐恢复

### 6. 摄像机控制 (Camera Control)
- 跟随集矿机视角
- 鼠标右键旋转视角
- 滚轮缩放
- Shift键切换自由漫游模式
- Q/E键升降高度

## 操作指南

### 基本控制
| 按键 | 功能 |
|------|------|
| W / ↑ | 向前移动 |
| S / ↓ | 向后移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| 空格 | 开始/停止采矿 |
| ESC | 暂停/继续 |
| Shift | 切换自由视角模式 |
| Q | 自由视角-下降 |
| E | 自由视角-上升 |
| 鼠标右键 + 拖动 | 旋转视角 |
| 鼠标滚轮 | 缩放 |

### 界面说明
- **生态面板**: 显示各项环境指标
- **采矿面板**: 显示采矿统计和采集速率调节
- **设置面板**: 重置和暂停功能

## 安装和运行

### 环境要求
- Unity 2020.3 或更高版本
- .NET Framework 4.x

### 安装步骤
1. 克隆或下载此项目
2. 打开Unity Hub
3. 点击"Add"按钮，选择项目文件夹
4. 等待Unity导入完成
5. 打开 `Assets/Scenes/MainScene.unity` 场景
6. 点击Play按钮运行

### 快速开始
1. 运行场景后，系统会自动生成地形、集矿机和海洋生物
2. 使用WASD移动集矿机到结核矿点
3. 按住空格键开始采矿
4. 观察UI面板中生态指标的变化
5. 尝试调节采矿速率，观察对环境的影响

## 教育意义

本模拟旨在通过交互式体验，帮助用户了解:
1. 深海多金属结核采矿的基本过程
2. 采矿活动对海洋环境的潜在影响
3. 沉积物羽流对海洋生态系统的危害
4. 可持续采矿的重要性
5. 环境保护与资源开发的平衡
